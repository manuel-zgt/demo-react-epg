## EPG - DEMO
EPG with React

---

#### Framework / Tools:

* React
* Redux
* Typescript
* Monorepo (lerna & yarn).

#### commands :

* install application:  on base path : `yarn install`
* run application:  `packages/react-app/` : `npm run start`
